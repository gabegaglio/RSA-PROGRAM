# RSA Programs

This repository contains RSA cryptography implementations organized into separate branches.

## Branches

| Branch | Description |
|--------|-------------|
| **RSA-Class** | RSA implementations for coursework/class assignments |
| **RSA-Exper** | Experimental RSA work with GMP library and large prime generation |
| **RSA-Personal** | Personal RSA implementation with full encryption/decryption system |

## Quick Access

```bash
# Class implementation
git checkout RSA-Class

# Experimental work
git checkout RSA-Exper

# Personal implementation
git checkout RSA-Personal
```
